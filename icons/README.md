# Extension Icons

To complete the Chrome extension, you need to add icons in the following sizes:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Creating Icons

You can create icons using:

1. **Online Tools**:
   - [Canva](https://www.canva.com) - Free design tool
   - [Figma](https://www.figma.com) - Free design tool
   - [Favicon.io](https://favicon.io) - Quick favicon generator

2. **Design Tips**:
   - Use YouTube's red color (#FF0000) for brand consistency
   - Simple, recognizable design (e.g., play button + text summary icon)
   - Make sure it's visible at 16x16 size
   - PNG format with transparency

3. **Quick Placeholder**:
   For testing, you can use solid color squares or generate simple icons online.
